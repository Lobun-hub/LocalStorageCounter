# LocalStorageCounter

A simple web-based counter application built with HTML, CSS, and JavaScript. The project demonstrates how to increase, decrease, reset, save, and load a counter value using the browser's localStorage.

## Features
- Increment and decrement the counter
- Reset the counter to zero
- Save the current count to localStorage
- Load the previously saved count
- Keep the value available across page refreshes

## Project Files
- index.html: Main counter page using inline event handlers
- script.js: Counter logic with localStorage support
- styles.css: Basic styling for the UI
- index1.html: Alternative version using event listeners
- script1.js: Alternative JavaScript implementation

## How to Run
1. Open index.html in your browser.
2. Use the buttons to change the count.
3. Click Save to store the value and Load to retrieve it later.

## Technologies Used
- HTML
- CSS
- JavaScript
- localStorage

## Notes
This project is a small demo for learning basic DOM manipulation and browser storage in JavaScript.
